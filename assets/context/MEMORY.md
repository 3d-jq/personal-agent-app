# MEMORY

跨场景长期记忆。

> 写入原则：只能记录用户明确说出来的事实，禁止推断、脑补，禁止写入用户未确认的分析或建议。
> 以下均为空模板，内容由模型在对话中逐步记录，禁止预填默认值。

## 长期稳定区

不易变化、持久有效，优先保留。

### 重要结论与决策

- [{日期}] {结论及理由}
- 说明：用户做出的重大决策和最终结论。一旦记录很少变动，除非用户推翻。

## 时效性区

随时间变化，定期更新或淘汰，超限时优先删除。

### 进行中的项目

- {项目名}（{日期}）
  - 目标：{值}
  - 进展：{值}
  - 待办：{值}
- 说明：活跃项目的当前状态。项目完成或长期无更新时应归档或删除。
