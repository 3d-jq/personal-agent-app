多厂商 AI

不锁定单一模型。任意兼容 OpenAI / Anthropic API 的服务商都可接入，独立配置、独立计费。

流式响应逐 token 实时渲染，打字机效果。支持 thinking / reasoning 模型的思考过程透明展示。
工具调用Function Calling 驱动 40+ 工具，大模型自主选择何时调用、调哪个、怎么传参。
上下文窗口支持 256K token 上下文，80% 阈值自动压缩，Prompt Caching 前缀命中减少重复计费。
用量追踪输入 / 输出 / 缓存命中 Token 实时统计，按厂商 + 模型 + 单价独立核算成本。

浏览器

内置 WebView，悬浮窗 + 工具双模式。大模型能真正"看见"网页内容。

完整操控导航、点击、输入、滚动、悬停、选择、表单填充——全部可编程。
视觉感知截图直接嵌入对话气泡，大模型分析页面布局和内容后给出操作建议。
文本提取页面快照、正文提取、元素搜索——配合搜索工具可实现自动调研。
灵活配置自定义 User-Agent、Viewport、Cookie——模拟任意设备和登录态。

Agent 群聊

多个 AI Agent 同场协作。每人独立人格、独立工具集、独立 AI 后端。

混合调度

既支持 @ 某个 Agent 点名回复，也支持自动推举最合适的 Agent 发言。最多 5 轮接力。

身份隔离

每个 Agent 只看得到自己与用户的对话，其他 Agent 的发言作为上下文注释——不串身份。

自定义 Agent

自由创建 Agent，独立配置 system prompt、工具白名单、模型选择。就像给团队配人手。

记忆系统

AI 不只是每次从零开始。它记得你、知道你的偏好、能跨会话延续上下文。

SOUL.mdAI 人格设定——名称、语气、风格、背景。定义"它是谁"。
USER.md用户资料——称呼、偏好、习惯。每次对话自动注入 system prompt。
MEMORY.md跨会话长期记忆——重要决定、项目约定、过往事实。压缩时合成摘要。
首次见面新用户触发自然引导：聊聊天就把称呼、名字、风格偏好收集好了。
